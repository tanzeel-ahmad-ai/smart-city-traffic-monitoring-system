# Smart City Traffic Monitoring System

An AI-powered traffic monitoring system designed to analyze real-time traffic flow and support smart city management.

## Features

- Real-time vehicle detection using YOLOv8
- Traffic density monitoring
- Route analytics and visualization
- Live traffic monitoring dashboard
- User authentication
- Interactive React.js frontend
- FastAPI backend integration
- MySQL database support

## Technologies Used

### Frontend
- React.js
- JavaScript
- HTML/CSS

### Backend
- FastAPI
- Python

### AI / Machine Learning
- YOLOv8
- OpenCV

### Database
- MySQL
